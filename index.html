Example 1: Teams + SharePoint collaboration

Evidence: screenshot of channels, shared files, or folder structure.

Screenshot here

Reflection (about 200 words): I learned that collaboration works best when the team has a clear structure. In Teams, creating separate spaces for chat, tasks, and final decisions reduced confusion. Using SharePoint for files helped avoid multiple versions being sent as attachments. I also understood how naming files and saving them in the right folder makes work easier to find later. A key takeaway was that version history and permissions are important: they protect files and show what changed. In a workplace, this supports accountability and makes handovers smoother. Overall, this example shows how I can use collaboration tools in a professional way, not just for messaging.

Example 2: Excel basics (clean and organised data)

Evidence: screenshot of your dataset before/after cleaning.

Screenshot here

Reflection (about 200 words): This task taught me that good spreadsheets start with clean data. When headings are clear and each column has one meaning, formulas and filters work properly. I practised formatting, consistent data types (numbers, dates), and simple validation to prevent mistakes. I realised that many Excel errors happen because data is messy, not because the math is difficult. In a workplace setting, a tidy spreadsheet saves time and makes it easier for another person to understand the file. This skill supports reliable reporting and better decision-making.

Example 3: Excel functions (IF / XLOOKUP / SUMIFS)

Evidence: screenshot showing formulas and outputs.

Screenshot here

Reflection (about 200 words): Using functions helped me turn raw data into useful information. IF statements allowed me to label results automatically. XLOOKUP helped me pull correct details from a reference table without retyping. SUMIFS helped me total values based on conditions. The main lesson was that Excel can behave like a small system when formulas are planned properly. In a workplace, this reduces manual work and improves consistency because the same rule is applied every time. I also learned to check formulas with test examples to make sure results are correct.

Example 4: PivotTables (fast summaries and insights)

Evidence: screenshot of PivotTable and filters/slicers.

Screenshot here

Reflection (about 200 words): PivotTables helped me summarise data quickly without building many formulas. I could compare categories, see totals, and filter results to answer questions faster. I learned that PivotTables are useful for reporting because the same dataset can create different views. Adding filters made the report easier to use. In a workplace, PivotTables support quick analysis for meetings, performance tracking, and decision-making. This example shows my ability to create a clear summary from a larger dataset.

Reflection (about 200 words): Creating digital content taught me to communicate clearly using layout, headings, and spacing.
